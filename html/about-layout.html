Privacy

Fire Tasks uses the Google Tasks API. To interact with the API, the user needs to be authenticated and request an "API KEY" for the app. The app stores this API key in the device's local storage â€” no one could access your API key, without firstly somehow accessing your device and/or your login and password. Developers do not use, distribute or in other ways publish these or any other sensitive data of the authenticated user.

Google Analytics is used to collect anonymous stats about app usage.

Anything you do while using Fire Tasks is absolutely up to you. Fire Tasks is completely open-source, so feel free to inspect source code if you meet any unusual behaviour and do not blame the developers.
